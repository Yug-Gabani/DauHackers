# Coastal Threat Alert System - CoastalGuard

A comprehensive, professional frontend website for a coastal threat alert system designed to protect coastal communities through real-time monitoring, early warnings, and community engagement.

## 🌊 Overview

CoastalGuard is an early warning system for storm surges, coastal erosion, and environmental threats. The platform saves lives and livelihoods through real-time alerts and community empowerment, serving various coastal communities across India.

## ✨ Key Features

### 🚨 Real-Time Alert System
- **Live Status Monitoring**: Real-time safety indicators for beachgoers and coastal communities
- **Multi-Level Alerts**: Critical, Warning, and Info level notifications
- **SMS Integration**: Instant SMS notifications without requiring smartphones or internet access
- **Multi-Language Support**: Alerts available in 8 local languages (English, Hindi, Marathi, Gujarati, Tamil, Telugu, Malayalam, Kannada)

### 📊 Interactive Dashboard
- **Live Weather Data**: Real-time weather conditions and forecasts
- **Tide Information**: Current tide levels and predictions with visual charts
- **Community Reports**: User-submitted reports on illegal activities, pollution, and coastal changes
- **Status Indicators**: Color-coded safety status (Green/Red) for different coastal areas

### 👥 Community Engagement
- **User Registration**: Comprehensive sign-up system with user type categorization
- **Community Reporting**: Citizens can report illegal activities, pollution, and unusual coastal changes
- **User Benefits**: Tailored information for different user groups (Fisherfolk, Business Owners, Tourists, NGOs, Authorities, General Public)

### 🛡️ Safety Features
- **Emergency Hotline**: Direct contact information for urgent situations
- **Evacuation Routes**: Real-time evacuation planning and shelter information
- **Preventive Alerts**: Early warnings for potential threats before they become dangerous

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Open `index.html` in a web browser
3. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```
